Demo — synthetic CDC producer + adaptive consumer

The demo runs a producer that writes synthetic CDC commits, and an adaptive consumer that processes them at a tunable batch size. It exposes its own dashboard when launched. Run from the CLI: 

riffle demo \
  --table-uri abfss://<container>@<account>.dfs.core.windows.net/path/demo.delta \
  --producer-enabled \
  --consumer-enabled \
  --bind-addr 127.0.0.1:8080

Embedding the demo in this dashboard is on the roadmap — it shares state with this UI but its job lifecycle is currently CLI-driven.
